# Application Quick Start

Use these files when applying:

1. **CV_POLISHED.md** - current recruiter CV.
2. **APPLICATION_MESSAGES.md** - message templates for Lemon.io, fractional advisor roles, cold outreach, and platform bios.
3. **PROJECTS.md** - deeper project evidence.
4. **WORK_PREFERENCES.md** - availability and fit boundaries.

Suggested first line for recruiters:

> I build practical systems: Python automation, LLM workflows, governance tooling, evidence systems, and security-aware prototypes.
